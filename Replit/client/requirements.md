## Packages
framer-motion | For smooth layout transitions and status animations

## Notes
The application uses a polling mechanism for research task updates.
The design focuses on a "Vault" aesthetic with a clean, scientific dashboard look.
Sidebar navigation is used for the main layout.
Detailed citations are displayed using an accordion style.
